# Module Template

Use this folder as the structural starting point for new business modules in `internal/modules/`.

Read first:

- `docs/MODULE_STANDARDS.md`
- `docs/PLATFORM_BOUNDARIES.md`
- `docs/READMODEL_AND_EVENTS_RULES.md`
- `docs/MODULE_CREATION_CHECKLIST.md`

Template layout:

- `domain/`
- `application/`
- `ports/`
- `adapters/`
- `transport/`
- `events/`
- `readmodel/`
