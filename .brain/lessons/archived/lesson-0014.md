# Lesson 0014 — Keep this file high-signal

**Date:** 2026-03-23 | **Trigger:** baseline | **Layer:** Process

## Correct
Record only recurring, cross-cutting, architectural, or process-level rules.

## Rule
If a lesson is not reusable across modules, it belongs in feature notes, not here.

**Medium:** Prevents dilution of the lessons knowledge base.
